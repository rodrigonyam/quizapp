## Image Sources

All images should be:
- Educational/public domain or properly licensed
- High quality (at least 800x600 pixels)
- Clear and readable for quiz purposes
- Properly attributed to their sources

## Adding Images

1. Download appropriate educational diagrams from the credited sources
2. Rename them to match the filenames in the quiz questions
3. Ensure they are in JPG format
4. Place them in this `/images` folder

## Placeholder Solution

If you don't have access to the actual images, you can:
1. Use placeholder images with the same filenames
2. Create simple diagrams using drawing software
3. Download similar educational diagrams from open sources like:
   - Wikimedia Commons
   - NASA image galleries
   - USGS educational resources
   - Educational institution websites
